# JS-Back-End-2026

## Cat Shelter

This is the introductory project for SoftUni's JS Back-End course. It serves as a first hands-on experience building a simple, dynamic, and functional web application using Node.js as the runtime environment, following a standard back-end workflow.

The project covers core back-end concepts such as routing, server-side rendering, and working with data — all while trying to follow best practices in code organization and structure.

> **Note:** This project is for educational purposes only.
